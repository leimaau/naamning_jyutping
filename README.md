# 安卓端配置文件

此處存放配置同文輸入法需要的文件，先安裝[小鶴同文主題包](http://flypy.ys168.com/)（選擇本分支下trime.yaml對應的主題版本），後用本分支下的文件覆蓋即可（包括trime.yaml）。

若嫌手工配置麻煩，[**此處**](https://github.com/leimaau/naamning_jyutping/releases)已通過 apktool 重新打包，下載安裝即可。
